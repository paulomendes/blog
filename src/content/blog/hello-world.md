## A Simple Swift Example

To kick things off, here's a classic "Hello, World!" in Swift:

```swift
import Foundation

struct Greeting {
    let message: String

    func display() {
        print(message)
    }
}

let greeting = Greeting(message: "Hello, World!")
greeting.display()

// Using Swift's modern features
let greetings = ["World", "iOS", "Swift"]
greetings.forEach { name in
    print("Hello, \(name)!")
}
```

Swift has come a long way, and I'm constantly amazed by how expressive and safe the language has become. From value types to protocol-oriented programming, there's always something new to explore.
